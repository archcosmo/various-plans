Various Plans - Information

Log on screen To use our application, you must login using your Google Account details. You will need to create one if you do not already have one. Create Plan Once successfully logged in, a new plan can be created. The name of the plan, the time at which voting and selecting plans and routes and the time of the actual event will need to be specified. Join Existing Plan A list of all previously joined plans can be seen. To continue to plan or view an existing plan, click on the tile of the plan in which you would like to visit. To join a plan, obtain the code provided by the creator and input it into the bottom of the popup screen. Upon successful addition of the join code, the plan will appear on your list of plans where it can now be selected. Stage 1 - Deciding Locations To search for locations in which you would like to visit, use the search bar on the map. Locations can be added to the list of locations by selecting the pin in which the map returns. From here you will be prompted to name the location and after it has been successfully added it will be added to the list in which userâ€™s part of the plan can now vote on. You can also vote on any of the locations suggested by other users by selecting the up or down arrow on each event. Once the end time of the stage has been reached (specified by the creator of the plan), all of the positively voted locations by users will be carried forward to Stage 2. Stage 2 - Deciding Routes: During this stage, the events carried forward from Stage 1 can now be ordered to create different routes in which you would like to take. To add a route, drag locations form the events table and place them into the route table. The order in which the locations are placed, is the order in which you would like to visit each location. To see each user suggested route, by hovering over each route, the route will be drawn on the map. Once again routes can be voted on by selecting the up or down arrows for each route. Final Stage Once the time has elapsed for Stage 2, the route with the highest number of votes (or a random route will be selected if joint highest) will be displayed to all users. The stops of the route can be seen and a visualisation of the route can be seen on the map.
