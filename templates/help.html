Various Plans

What is Various Plans?

Various Plans is an event planning application that allows you and your friends to collaboratively design your perfect meet up or day out with the use of our unique voting system.

Accounts

We appreciate that people do not like creating accounts for all the services they use and registeration processes can be tedious. Therefore, to use our application, you can login using your Google Account details.

How it Works?

Stage 1 - Deciding Locations

In this stage, all the people in the group can add suggestions for places that they may think are relevant to the event. Everyone can then vote on all the suggested locations; up-voting if they like the suggestion or down-voting otherwise. This can be done until the end time of the stage has been reached (specified by the creator of the plan), all of the positively voted locations by users will be carried forward to Stage 2.

To search for locations which you would like to suggest, use the search bar on the map. Locations can be added to the list of locations by selecting the pin in which the map returns. From here you will be prompted to name the location and after it has been successfully added it will be added to the list in which userâ€™s part of the plan can now vote on. You can vote on any of the locations suggested by other users by selecting the up or down arrow on each event. You can click on different locations to view more details about the location.

Stage 2 - Deciding Routes

Now that we have the list of events people are positively interested in, all the poeple in the group can now suggest routes that can be taken for the trip. Like in Stage 1, everyone can then vote on all the suggested routes; up-voting if they like the suggestion or down-voting otherwise. This can be done until the end time of the stage has been reached (specified by the creator of the plan).

To add a route, drag locations form the events table and place them into the route table. The order in which the locations are placed, is the order in which you would like to visit each location. To see each user suggested route, by hovering over each route, the route will be drawn on the map. Once again routes can be voted on by selecting the up or down arrows for each route. Clicking on a route gives you more details about the route.

Once the time has elapsed for Stage 2, the route with the highest number of votes (or a random route will be selected if joint highest) will be displayed to all users. The stops of the route can be seen and a visualisation of the route can be seen on the map.

Create A Plan

Once successfully logged in, a new plan can be created. From the Home Page you can click Create a Plan. You will need to enter the following details:

  • Name of the Plan - This is the name of your plan that can you can use to identify your plans easily when you choose to view your plans
  • Events Close Time (Stage 1) - This is the time when you would like to end the voting and the suggestions of places you and your friends would like to visit. Please see Stage 1 for more details.
  • Routes Close Time (Stage 2) - This is the time when you would like to end the voting and the suggestions of routes you and your friends would like to create. Please see Stage 2 for more details.
  • Time of Plan - This is the time of when your plan is taking place.

View or Join an Existing Plan

From the Home Page you can click View or Join an Existing Plan. Here, you can view the plans you have already join or join a new plan using a join in code. You can then click on an event to view it.
